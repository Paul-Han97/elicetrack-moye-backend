### 프로젝트 우수상 수상

### 개발 기간
- 2024년 10월 07일 ~ 2024년 10월 21일

### 서비스 내용
- 가게 사장님이 사용하시는 매장 예약 관리 서비스

### 개발 인원
- 프론트엔드 3명
- 백엔드 1명 (본인)

### 구현 기능
- 여러개의 매장을 추가하여 관리하는 기능
  - 평일 운영시간 / 휴식시간
  - 주말 운영시간 / 휴식시간
  - 정기 휴일 / 비정기 휴일
  - 매장 정보(상호명, 사업자 번호, 매장 전화번호, 이메일, 좌석 수, 최대 예약 가능 인원 등)
- 손님이 예약한 시간대를 타임라인 및 월간 예약 페이지로 확인하는 기능
- 확정, 대기, 취소 등 3가지의 상태를 가진 예약을 수정하는 기능
